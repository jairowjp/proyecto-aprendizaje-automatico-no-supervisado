# Proyecto 3 - Segmentación de clientes con aprendizaje no supervisado

## Objetivo
Implementar y analizar modelos de aprendizaje no supervisado (K-means, DBSCAN, PCA y t-SNE) para segmentar clientes a partir de datos de navegación, transacciones y comportamiento.

## Estructura del proyecto
- `data/raw/`: datos originales
- `data/interim/`: datos intermedios
- `data/processed/`: datasets procesados y features
- `notebooks/`: notebooks del análisis paso a paso
- `src/`: funciones reutilizables
- `reports/figures/`: figuras exportadas
- `reports/tables/`: tablas resumen
- `slides/`: presentación final

## Flujo de trabajo
1. Preparación del entorno
2. Carga y validación de datos
3. Análisis exploratorio
4. Feature engineering
5. K-means
6. DBSCAN
7. PCA y t-SNE
8. Conclusiones
